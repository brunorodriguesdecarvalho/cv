  1. O que é um Scorecard

    • Origem: Inglês
    • Tradução: Tabela de Desempenho
    • Para que serve: Monitorar o desempenho de algo que é importante.
    • Exemplo mais conhecido: Boletim escolar.

  2. O que é um Balanced Scorecard

    • Criadores: Robert Kaplan e David Norton
    • Data da criação: 1992
    • Para que serve: Trata-se de uma abordagem para gerenciar o planejamento estratégico.
    • Abreviação: BSC
    • Wiki: Clique aqui

  3. O que é um Hub

    • Origem: Inglês
    • Tradução: Eixo, Pivô, Central
    • Definição: A parte central ou a maior parte de algo onde ocorre uma atividade específica.
    • Exemplo 1: O aeroporto de Viracopos é utilizado pela Azul Linhas Aéras como hub, de onde a maior parte dos vôos da empresa são originários.
    • Exemplo 2: O processo pelo qual se transmite ou difunde determinada informação, tendo, como principal característica, que a mesma informação está sendo enviada para muitos receptores ao mesmo tempo (broadcast). Este termo é utilizado em rádio, telecomunicações e em informática.
