  1. Resumo sobre a Linguagem

    • É utilizado para:
      • Desenvolvimento Web (server-side): Criar aplicações
      • Sesenvolvimento de Software (ex: utilizar em conjunto com Workflow)
      • Matemática e estatística: Manipulação de Big Data e cálculos complexos.

    • Por que Python?
      • Multiplataforma (Win/Mac/Linux...)
      • Sintáxe simples e muito parecida com inglês
      • O código costuma ser mais enxuto.
      • Tem suporte a diferentes paradigmas de progrmação (procedural, orientada a objetos e funcional).

    • Versão atual: A versão mais atual é a 3. Não há mais suporte a versão 2 apesar dela continuar popular.
    • IDE: Python pode ser escrito em uma IDE, como Thonny, Pycharm, Netbeans ou Eclipse.
